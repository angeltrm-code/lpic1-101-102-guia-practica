# 04 — Matriz comando → objetivo (LPIC-1 101 y 102)

> Documento de trabajo para el proyecto LPIC-1.
> Reorganiza los elementos del temario desde la perspectiva inversa: **cada comando, archivo o utilidad** y los **objetivos** donde aparece o encaja.

## Cómo leer este documento

- **Objetivos oficiales:** el comando o elemento aparece citado en la web oficial de objetivos de LPI.
- **Objetivos asociados:** no necesariamente aparece de forma explícita en la web, pero es útil para practicar ese objetivo.
- Este archivo sirve para repasar por comando en vez de repasar por tema.
